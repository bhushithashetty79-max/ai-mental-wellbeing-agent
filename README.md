# 🧠 AI Mental Wellbeing Agent

An AI-powered Streamlit application that provides personalized mental wellbeing insights based on mood, sleep, and stress levels.

## ✨ Features
- Mood, Sleep & Stress sliders
- Personalized AI-generated support
- Beautiful glassmorphism UI
- Secure API key input
- Powered by Groq LLaMA 3.1 8B

## 🛠 Tech Stack
- Python
- Streamlit
- Groq API
- LLaMA 3.1-8B-Instant

## 🚀 How to Run

1. Clone the repository
2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run ai_mental_wellbeing_agent.py
```

4. Enter your Groq API key in the sidebar.

---

⚠ Note: Do not upload your API key publicly.
